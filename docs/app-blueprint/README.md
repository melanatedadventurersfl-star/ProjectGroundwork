# Melanated Adventurers App Blueprint

This folder is the product source of truth for the Melanated Adventurers member platform.

## Purpose

The blueprint defines what the platform is, why it exists, how the member experience is organized, and how future product decisions should be made.

## Prototype Route

The first interactive build should support:

```text
Trailhead
→ Explore
→ Adventure Detail
→ Registration
→ Confirmation
→ Adventure Hub
→ Check-In
→ Passport Stamp
→ Journey Entry
```

## Guiding Statement

> Help people build meaningful relationships through shared outdoor experiences.

## Document Practice

Each chapter should identify its current decision, open questions, and decision history. These documents are expected to evolve as member research, prototypes, and real-world testing produce better evidence.
