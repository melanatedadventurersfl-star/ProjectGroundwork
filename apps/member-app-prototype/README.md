# Melanated Adventurers Member App Prototype

This folder contains the first interactive prototype for the Melanated Adventurers member experience.

## Prototype route

Trailhead → Explore → Adventure Detail → Registration → Confirmation → Adventure Hub → Check-In → Passport

## Run locally

Open `index.html` in a browser. No build step is required.

## Scope

- Mobile-first interface
- Metro-inspired live tile dashboard
- Clickable navigation
- Adventure discovery and registration flow
- Adventure preparation hub
- Check-in confirmation
- Passport stamp and Journey entry

This is a front-end prototype using sample data only. It does not yet include authentication, payments, persistent storage, or a backend.
